# Attendance Record

A Laravel-based attendance registration system with Docker support. Features public event registration with digital signature capture, a Filament admin panel, QR code generation, and CSV reporting.

## Tech Stack

- **Backend:** Laravel 13 (PHP 8.3)
- **Frontend:** Tailwind CSS v4, Vite, Signature Pad
- **Admin Panel:** Filament v5
- **Database:** MySQL 8.4
- **PDF/Reports:** DOMPDF, FPDF, PhpSpreadsheet
- **QR Codes:** Bacon QR Code
- **Containerization:** Docker & Docker Compose

## Services

| Service | Image / Build | Port | Description |
|---------|---------------|------|-------------|
| `app` | PHP-FPM (custom) | `9000` (internal) | Laravel application |
| `nginx` | `nginx:alpine` | `8080` | Web server / reverse proxy |
| `db` | `mysql:8.4` | `3307` | MySQL database |
| `phpmyadmin` | `phpmyadmin:5.2-apache` | `8081` | Database admin UI |
| `vite` | Node (custom) | `5173` | Vite dev server with HMR |

## Quick Start

### Prerequisites

- Docker Engine & Docker Compose
- Make (optional, for convenience commands)

### Setup

1. Copy environment variables and start the services:

   ```bash
   cp .env.example .env
   docker compose up -d
   ```

2. Run migrations and seed the database:

   ```bash
   docker compose run --rm app php artisan migrate
   docker compose run --rm app php artisan db:seed
   ```

3. The application is available at:

   - **App:** http://localhost:8080
   - **Vite HMR:** http://localhost:5173
   - **phpMyAdmin:** http://localhost:8081

### Default Admin Credentials

- **Email:** `admin@example.com`
- **Password:** `admin123`

> Generated by the database seeder. Change these immediately in production.

## Key Features

- **Public Event Registration:** Attendees register via a public form (`/evento/{slug}`) with digital signature capture.
- **Admin Dashboard:** Filament-powered admin panel for managing events, attendance records, positions, headquarters, and reasons.
- **QR Code Generation:** For event check-in or quick access links.
- **CSV Export:** Attendance reports filtered by event (`/admin/reportes/asistencias?event_id=`).
- **PDF Reports:** Generated with DOMPDF and FPDF.
- **Spanish UI:** All interface labels and validation messages in Spanish.
- **Soft Disabling:** Positions, headquarters, and reasons can be soft-disabled via `is_active` boolean.

## Common Commands

### Docker

```bash
# Start all services
docker compose up -d

# Stop all services
docker compose down

# View logs
docker compose logs -f [service]

# Run artisan commands
docker compose run --rm app php artisan [command]

# Run composer
docker compose run --rm app composer [command]

# Run tests
docker compose run --rm app php artisan test

# Build assets (production)
docker compose run --rm vite npm run build
```

### Inside the `app` Container

```bash
docker compose exec app bash

# Then run any artisan/composer/npm command directly
php artisan migrate:fresh --seed
composer install
npm run dev
```

## Environment Variables

Key variables in `.env`:

| Variable | Default | Description |
|----------|---------|-------------|
| `DB_DATABASE` | `laravel` | MySQL database name |
| `DB_USERNAME` | `laravel` | MySQL user |
| `DB_PASSWORD` | `laravel` | MySQL password |
| `DB_ROOT_PASSWORD` | `root` | MySQL root password |
| `APP_USER` | `laravel` | System user inside the app container |
| `UID` / `GID` | `1000` | Host user/group IDs for file permissions |

## Customization

### Admin Panel Theme

The Filament admin panel theme is customized via:

- `src/app/Providers/Filament/AdminPanelProvider.php` вЂ” brand name, colors, fonts, theme registration.
- `src/resources/css/filament/admin/theme.css` вЂ” custom CSS overrides for Filament components.

### Public Pages Theme

Public pages use custom Tailwind colors defined in `src/resources/css/app.css` (`navy-*`, `gold-*`, `warm-*`). These are independent from the admin theme; redefine them in both files if needed.

## Testing

```bash
docker compose run --rm app php artisan test
```

PHPUnit is configured with an in-memory SQLite database (`phpunit.xml`). Test suites: Unit, Feature.
